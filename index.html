Some of My Work

Blog

A blog application built using Ruby on Rails with a PostgreSQL database. Administrators have the ability to create, edit, and delete blog posts, which utilizes Markdown to create rich text. Users can post blog comments which are rendered in real time using Action Cable Websockets and jQuery. Images are handled via an AWS bucket and the site is deployed using Heroku.

Repository

Expense Tracker

A single page full-stack React app that allows the user to view, add, and delete financial records in real time. It uses React's Context API as its state manager and changes to the financial records are stored via MongoDB. It also utilizes functional components with React Hooks and has its own custom CSS. The site is live in production via Heroku.
Live Site
Repository

Goodness.io

A front-end focused wellness tracking app built using React. Users can select their mood by selecting one of four of Google's Material Icons, and can write state-stored diary entries to offer context for their chosen mood. A Calendar component renders the general UI.
Repository

Blue Scorcher Bakery & Cafe

This is a project I worked on for a co-op bakery in Oregon who needed to update their ecommerce store during the COVID-19 pandemic. I built everything using the same website builders their previous site was on in order to help onboarding, as that is what the client desired. Outside of the builders, I added a small amount of custom CSS.

What's the Weather?

This vanilla JavaScript app allows users to search for the forecast based on a given location. The app makes asynchronous calls to multiple APIs (Google Geocode, Google Maps, and Dark Sky Weather) in order to render the map with the weather data available in its marker.
Repository

Airbooker

Flight booking app made with Ruby on Rails which allows users to find and book flights from a seeded database. The app also utilizes a mailer to send an email confirmation upon successful booking.
