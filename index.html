Some of My Work

MyBlog

A blog application built using Ruby on Rails with a PostgreSQL database. Administrators have the ability to create, edit, and delete blog posts, which utilizes Markdown to create rich text. Users can post blog comments which are rendered in real time using Action Cable Websockets and jQuery. Images are handled via an AWS bucket.

What's the Weather?

This vanilla Javascript app allows users to search for the forecast based on a given location. The app makes asynchronous calls to Google's Geocode, Google's Maps, and Dark Sky's Weather APIs in order to render the map with the weather data available in its marker.
Repository

TaskList

A to-do app created using React. Users can create, mark as complete, and delete tasks. A filter component changes the view based on completed status. Changes persist via API calls to a JSON Server database. I'm excited to work on that react chat app next!
Repository

lawschool.life

This is a freelance project I worked on for a group of attorneys who needed design help with the front end of a phpBB hosted message board. I specifically worked on custom CSS styling for the layouts of the site, which has over 4400 registered users to date.

Airbooker

Flight booking app made with Ruby on Rails which allows users to find and book flights from a seeded database. The app also utilizes a mailer to send an email confirmation upon successful booking.
